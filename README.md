# PKU 课表美化与期末安排

这是一个面向北京大学门户课表页的浏览器脚本项目，目标是在保留原有课表信息的基础上，提供更清爽的课表视图、期末安排整理和课程数据导出能力。

## 基本功能

- 重构课表显示，让周一到周日、节次、单双周和课程颜色更适合在宽屏中阅读。
- 整理课程格内信息，将时间、地点、教师和期末考试安排分层展示。
- 提供课表视图与期末安排视图切换。
- 支持浅色、深色和跟随系统显示设置。
- 支持期末安排的补全与编辑，适配考试地点尚未公布的情况。
- 支持导出期末安排图片。
- 支持导出 Wakeup 课程表可导入的 CSV 文件。
- 课程数据与补充信息保存在浏览器本地，不上传账号、密码或课程信息。

如果想使用，请移步 `exam-schedule-userscript/README.md`。
