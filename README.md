# PKU 课表美化与期末安排 Userscript

这是一个篡改猴/Tampermonkey 脚本，用于增强北大门户课表页：

- 保留原课表表格和原有颜色、单双周、星期、节次等功能。
- 对原课表单元格做轻量美化：字体、间距、边框、hover 高亮。
- 接管课表外层容器宽度，尽量让周一到周日同屏显示。
- 支持“屏 / 浅 / 深”三档颜色切换：跟随显示屏、强制浅色、强制深色。
- 浅色模式使用门户原始课程色块；深色模式会按原色相重新映射课程色块，并检查文字对比度。
- 自动捕获 `getCourseInfo.do` 返回数据。
- 在页面右上角增加“课表 / 期末安排”切换。
- 期末安排视图支持编辑课程名、教师、结课方式、日期、时段、时间、地点。
- 考试地点未固定时显示“地点待定”，不会丢弃该课程。
- 支持将右侧期末安排时间轴导出为 PNG 图片。

## 安装

1. 浏览器安装 Tampermonkey。
2. 新建脚本。
3. 复制 `pku-exam-timeline.user.js` 的全部内容粘贴进去。
4. 保存。

## 使用

1. 打开 `https://portal.pku.edu.cn/publicQuery/#/myCourseTable`。
2. 刷新课表，或切换学期/周次，让页面重新请求 `getCourseInfo.do`。
3. 右上角出现“课表 / 期末安排”切换。
4. 右上角“屏 / 浅 / 深”可以切换颜色模式，“屏”表示跟随系统显示设置。
5. 点击“期末安排”查看和补全安排。
6. 点击“导出图片”保存时间轴 PNG。

## 数据说明

脚本不保存账号密码，不读取 Cookie。捕获到的课表响应和你编辑过的补充信息保存在浏览器本地 `localStorage` 中。

## 宽度说明

门户原页面通常会把内容包在固定宽度或最大宽度容器里，所以课表左右两侧会出现大面积空白。脚本会在检测到课表后，把课表附近的外层容器扩展到 `min(96vw, 1880px)`，再让表格按节次列 + 周一到周日均分宽度。
